"# Stream"

note run from Git Bash terminal: "./run_dev.sh" to run full app

---
Content Discovery & Navigation

Predictive/Fuzzy Search: Handling typos (e.g., "Spidrman" -> "Spider-Man").
"Trending" or "Popular" Lists: displaying what everyone else is watching right now.
Recommendation Engine: "Because you watched X, you might like Y."

User Personalization

Watchlist / "My List": A place to save movies to watch later.
Favorites: A way to "heart" content you love.
Watch History: A log of what the user has already seen.
Multiple Profiles: Separate watch histories/lists for different people in the same household.
Parental Controls: PIN protection or age restrictions for kids' profiles.

Media Presentation (The "Card")

Poster & Backdrop Art: High-quality visual assets.
User Rating: The 1-10 score or Star rating.
Trailers: An embedded video player to watch the trailer.

Video Playback Features

Play/Pause/Seek: Basic controls.
Skip Intro / Skip Recap: A button to bypass opening credits.
Auto-Play Next Episode: Automatically starting the next episode after a countdown.
Subtitles & Audio Tracks: Changing languages or enabling Closed Captions (CC).
Video Quality Selection: Options for 4K, 1080p, 720p, or "Data Saver".
Playback Speed: Watching at 0.75x, 1.25x, or 2x speed.
Picture-in-Picture (PiP): Watching the video in a small floating window while using other apps.

Technical & Utility

Cross-Platform Sync: Start searching on your phone, finish on your desktop.
Offline Downloads: Downloading content to watch without internet.
